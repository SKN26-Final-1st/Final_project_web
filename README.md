# web_design_playground

| | |
|---|---|
| **팀** | 육뚝이들 |
| **프로젝트** | **HumouR** |
| **과정** | SKN26 파이널 프로젝트 |

SKN26 파이널 프로젝트 **HumouR**의 **UI 목업·디자인 연습** 저장소입니다.  
실제 백엔드·인증·데이터 연동 없이, 화면 구성·레이아웃·컴포넌트 배치·라이트/다크 테마 등 **프론트엔드 UI를 빠르게 시도해 보는 샌드박스** 역할을 합니다.

## 목적

- **HumouR**(채용·지원자 분석 관련 서비스)의 화면 흐름을 미리 그려 보기
- Ant Design 기반으로 카드, 테이블, 폼, 대시보드 등 **실사용에 가까운 목업** 연습
- 본 레포에서 검증한 레이아웃·토큰·패턴을 이후 메인 프로젝트에 옮기기

## 포함된 화면 (목업)

| 구분 | 화면 |
|------|------|
| 메인 | 대시보드, 회사 정보, JD 관리, 자기소개서, 채팅, 마이페이지, 채용 공고, 자소서 템플릿 |
| 인증 | 로그인, 회원가입, 비밀번호 재설정 |

모든 데이터는 `src/data/mockData.tsx`의 **목 데이터**이며, API 호출은 없습니다.

## 기술 스택

- React 19 + TypeScript
- Vite 7
- Ant Design 5

## 로컬 실행

```bash
npm install
npm run dev
```

브라우저에서 `http://127.0.0.1:5173` (또는 터미널에 표시된 주소)로 접속합니다.  
화면 이동은 URL **해시 라우팅**(`#/dashboard`, `#/jd` 등)을 사용합니다.

```bash
npm run build   # 타입 검사 + 프로덕션 빌드
npm run lint    # ESLint
npm run preview # 빌드 결과 미리보기
```

## 참고

- 이 레포는 **육뚝이들** 팀의 **디자인·UI 실험용**이며, HumouR 메인 프로젝트의 공식 백엔드/배포 저장소와는 분리되어 있습니다.
- QA 캡처 등 로컬 산출물은 `.gitignore`에 정의되어 있습니다 (`dist/`, `qa-screenshots/` 등).
